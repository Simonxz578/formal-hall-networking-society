# Formal Hall Networking Society

A static website for the Formal Hall Networking Society.

## Local preview

Run:

    python3 -m http.server 8000

Then open:

    http://localhost:8000

Hard refresh if needed:

    Command + Shift + R

## Project structure

    .
    ├── index.html
    ├── assets/
    │   ├── css/
    │   ├── js/
    │   └── images/
    └── README.md

## Notes

This project is designed as a static website and can be previewed locally with Python's built-in HTTP server.
